# 🚒 الحماية المدنية - ولاية ميلة

تطبيق PWA لدليل وحدات الحماية المدنية لولاية ميلة.

## 📁 هيكل الملفات

```
mila-pwa/
├── index.html       ← الصفحة الرئيسية
├── manifest.json    ← إعدادات PWA
├── sw.js            ← Service Worker (للعمل offline)
├── icons/
│   ├── icon-192.png
│   └── icon-512.png
└── README.md
```

## 🚀 طريقة النشر على GitHub Pages

### الخطوة 1 - إنشاء المستودع
1. افتح [github.com](https://github.com) وسجل دخولك
2. اضغط **New repository**
3. اسمه مثلاً: `hmaya-mila`
4. اجعله **Public**
5. اضغط **Create repository**

### الخطوة 2 - رفع الملفات
**الطريقة السهلة (بدون Git):**
1. افتح المستودع الجديد
2. اضغط **Add file → Upload files**
3. ارفع جميع الملفات (index.html, manifest.json, sw.js, ومجلد icons)
4. اضغط **Commit changes**

### الخطوة 3 - تفعيل GitHub Pages
1. اذهب إلى **Settings** في المستودع
2. من القائمة الجانبية اختر **Pages**
3. في **Source** اختر: `Deploy from a branch`
4. في **Branch** اختر: `main` ثم `/ (root)`
5. اضغط **Save**

### الخطوة 4 - الرابط
بعد دقيقتين سيكون التطبيق متاحاً على:
```
https://اسم-المستخدم.github.io/hmaya-mila/
```

## 📲 تثبيت التطبيق على الهاتف
- **Android**: افتح الرابط في Chrome → ستظهر رسالة "تثبيت التطبيق"
- **iPhone**: افتح الرابط في Safari → اضغط مشاركة → "إضافة إلى الشاشة الرئيسية"

## ✅ مميزات التطبيق
- يعمل بدون إنترنت (Offline) بعد أول زيارة
- قابل للتثبيت على الهاتف كتطبيق
- أرقام قابلة للنقر للاتصال المباشر
- واجهة عربية RTL داكنة
